# Kas-Flow

Open-source payment infrastructure for Kaspa blockchain.

## Projects

### [kasflow](https://github.com/Kas-Flow/kasflow)
Monorepo containing our SDK packages for building Kaspa applications.

## npm Packages

| Package | Description |
|---------|-------------|
| [@kasflow/passkey-wallet](https://www.npmjs.com/package/@kasflow/passkey-wallet) | Passkey-powered wallet SDK - create Kaspa wallets using device biometrics |
| [@kasflow/wallet-connector](https://www.npmjs.com/package/@kasflow/wallet-connector) | Multi-wallet connector with React hooks and components |

## Quick Install

```bash
npm install @kasflow/passkey-wallet
npm install @kasflow/wallet-connector
```
